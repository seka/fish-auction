# コーディング標準

## 全般

- インデントには 2 スペースを使用します（Go はタブを使用）。
- 変数名や関数名は意味のある適切な名称を優先します。
- 関数は小さく、単一の責任を持つように保ちます。

## TypeScript / JavaScript

- 厳格な型チェック（Strict mode）を適用します。
- `any` 型の使用を避け、インターフェースや型定義を使用します。
- Prettier でフォーマットし、ESLint でリンティングを行います。

## Go

- `gofmt` および `goimports` に従います。
- エラーハンドリングを明示的に行います。
- キャンセルやタイムアウトの管理には `context.Context` を使用します。
