# アーキテクチャガイドライン

## 概要

このプロジェクトは、GoバックエンドとNext.jsフロントエンドで構成されるウェブアプリケーションです。

## バックエンド (Go)

- 標準的なGoプロジェクトのレイアウトに従います。
- ルーティングには `gorilla/mux`、データベースアクセスには `database/sql` を使用します。
- ビジネスロジックは `internal/` パッケージ内に保持します。

## フロントエンド (Next.js)

- App Router を使用します。
- TypeScript を用いた関数コンポーネントで開発します。
- スタイリングには Panda CSS を使用します。

## 通信

- クライアント・サーバー間の通信には REST API を使用します。
